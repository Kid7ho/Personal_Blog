백준 1000. A+B
Jin-Myoung Hyun · Algorithms / Mathematics · 2023.11.19

백준 1000. A+B: https://www.acmicpc.net/problem/14940

문제:

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.


입력:

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)


출력:

첫째 줄에 A+B를 출력한다.

풀이:

2개의 입력을 받아 합을 출력하면 되는 단순한 문제이다. 입력으로 두 개의 정수가 공백으로 구분되어 주어지기 때문에 각 언어의 'split' 기능 이용해 두 정수를 분리하고 각각의 정수를 변수에 담아 합을 출력한다.

소스 코드:

  • C/C++
  • C#
  • Java
  • Python
  • Kotlin
  • Dart
  • JavaScript
